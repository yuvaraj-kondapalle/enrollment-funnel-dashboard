# Enrollment Funnel Dashboard

An interactive data visualization dashboard built with Python, Streamlit, 
and Plotly to analyze student progression across enrollment funnel stages.

## Overview

This dashboard tracks student movement from initial Inquiry through to 
Enrollment, providing actionable insights for enrollment management teams 
at higher education institutions.

## Features

- **Enrollment Funnel Visualization** — track drop-off across five stages: 
  Inquiry, Application, Admitted, Deposited, Enrolled
- **Program Breakdown** — applications segmented by academic program
- **Demographic Analysis** — domestic vs international student split
- **Financial Aid Distribution** — aid eligibility across the applicant pool
- **Interactive Filters** — filter by year, program, and demographics in real time

## Tech Stack

- Python
- Streamlit
- Plotly Express
- Pandas
- NumPy

## How to Run

1. Clone the repository
   git clone https://github.com/yuvaraj-kondapalle/enrollment-funnel-dashboard.git

2. Install dependencies
   pip install -r requirements.txt

3. Run the app
   streamlit run app.py

4. Open your browser at http://localhost:8501

## Use Case

Built to demonstrate applied enrollment analytics skills — funnel modeling, 
demographic segmentation, and data visualization for non-technical 
stakeholders — directly relevant to yield modeling and recruitment analytics 
in higher education.
